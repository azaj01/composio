The Mastra provider formats Composio tools into a format compatible with [Mastra](https://mastra.ai/en/docs/tools-mcp/overview#creating-tools)'s tool calling capabilities, including input schemas, execution logic, and descriptions.

## Setup

```bash
npm install @composio/core @composio/mastra @mastra/core @ai-sdk/openai
```

## Usage

```typescript
import { Composio } from "@composio/core";
import { MastraProvider } from "@composio/mastra";
import { Agent } from "@mastra/core/agent";
import { openai } from "@ai-sdk/openai";

const composio = new Composio({
  provider: new MastraProvider(),
});

// Create a session for your user
const session = await composio.create("user_123");
const tools = await session.tools();

const agent = new Agent({
  id: "github-agent",
  name: "GitHub Agent",
  instructions: "You are a helpful assistant that can perform GitHub actions.",
  model: openai("gpt-4.1"),
  tools: tools,
});

const { text } = await agent.generate([
  { role: "user", content: "Star the composiohq/composio repository on GitHub" },
]);

console.log(text);
```

## Usage with MCP

You can also use Composio MCP servers directly with Mastra's MCPClient.

<Callout type="info">
Composio MCP servers only support Streamable HTTP transport.
</Callout>

```typescript
// @noErrors
import { MCPClient } from "@mastra/mcp";
import { openai } from "@ai-sdk/openai";
import { Agent } from "@mastra/core/agent";
import { Composio } from "@composio/core";

const composio = new Composio();

// Create a session for your user
const session = await composio.create("user_123");

// Create MCP client with the session's MCP server
const mcpClient = new MCPClient({
  id: "composio-mcp-client",
  servers: {
    composio: { url: new URL(session.mcp.url) },
  }
});

const agent = new Agent({
  id: "dev-assistant",
  name: "Dev Assistant",
  instructions: "Help manage GitHub repos, Linear issues, and Notion documentation.",
  model: openai("gpt-4-turbo"),
  tools: await mcpClient.getTools()
});

const response = await agent.generate(
  "Star the composiohq/composio repository on GitHub"
);
console.log(response.text);
```
