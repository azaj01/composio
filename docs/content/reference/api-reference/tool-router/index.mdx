## Endpoints

| Method | Path | Name | Description |
|--------|------|------|-------------|
| POST | `/api/v3/tool_router/session` | [Create a new tool router session](/reference/api-reference/tool-router/create-a-new-tool-router-session) | Creates a new session for the tool router feature. This endpoint initializes a new session with specified toolkits and their authentication configurations. The session provides an isolated environment for testing and managing tool routing logic with scoped MCP server access. |
| POST | `/api/v3/tool_router/session/{session_id}/execute` | [Execute a tool within a tool router session](/reference/api-reference/tool-router/execute-a-tool-within-a-tool-router-session) | Executes a specific tool within a tool router session.
